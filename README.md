# GamineAI Help Center 🎮🤖

Welcome to GamineAI — your AI-powered game creation platform.

This help center is designed to guide you step by step on how to create games using artificial intelligence, improve results, and fix common issues.

---

## 🎮 What is GamineAI?

GamineAI is an AI game generator that allows you to create playable games instantly using simple text prompts.

Instead of coding, you describe your game — and the AI builds it for you.

---

## 🚀 How to Create a Game

Creating a game on GamineAI is simple:

1. Go to the **Create Game** page
2. Write a clear prompt (example: *"Create a 2D snake game with increasing speed"*)
3. Click generate
4. Play your game instantly

---

## 🤖 How the AI Works

GamineAI uses advanced AI models to:

* Generate game logic
* Build basic mechanics
* Create playable experiences

The better your prompt, the better your game.

---

## 💡 Tips for Better Games

To get high-quality results:

* Be specific (game type, mechanics, difficulty)
* Keep your prompt simple and clear
* Use known game styles (runner, puzzle, arcade)
* Try multiple variations

---

## ⚠️ Common Issues & Solutions

### Game not generating?

* Simplify your prompt
* Avoid long or complex descriptions

---

### Game is slow?

* Refresh the page
* Use Chrome or a modern browser
* Disable ad blockers

---

### Results are not good?

* Improve your prompt
* Add more details (speed, style, rules)
